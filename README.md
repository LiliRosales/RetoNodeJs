# RetoNodeJs

Crear un nuevo paquete:
    npm init

    Escribir el archivo para el punto de entrada    
    entry point:(index.js) server.js

    Crear un nuevo archivo
        touch server.js

    Instalar paquetes 
        npm install express

    Compilar para verificar funcionalidad 
        node server.js

    Instalar mongoose
        npm i mongoose
    
    Para verificar la version del node
        node -v
